EasySearch

简易搜索聚合工具

基于 Python + PySide6 开发的桌面搜索工具

聚合多个搜索引擎，智能过滤排序

项目概述

EasySearch 是一个基于 Python 和 PySide6 开发的桌面搜索聚合应用程序。它通过配置多个搜索引擎API，聚合搜索结果并进行智能排序和过滤。

说明：这是一个单文件 Python 应用程序，仅需要 icon.ico 作为图标文件。

核心设计理念

1. 简洁实用：界面简洁，功能实用，无冗余设计

2. 配置灵活：支持自定义多个搜索引擎API和过滤规则

3. 实时响应：多线程搜索，结果实时返回和显示

主要功能

  • 多搜索引擎聚合

    同时配置多个搜索引擎API进行并行搜索，支持自定义API URL、密钥和结果字段映射

  • 智能权重排序

    基于网站权威性、发布时间、黑白名单等因素自动计算权重并排序

  • 配置管理系统

    完整的设置界面，支持黑白名单管理、API配置、主题切换，所有设置自动保存

  • URL规范化处理

    自动去除跟踪参数(utm_*, fbclid等)，标准化URL格式，实现有效的搜索结果去重

  • 智能时间解析

    支持多种时间格式（时间戳、ISO格式、字符串日期）的发布日期间解析和显示

  • 主题系统

    提供浅色、深色和跟随系统三种主题模式，界面元素自动适配
